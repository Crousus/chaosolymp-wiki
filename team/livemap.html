Livemap Befehle

 

  • /dynmap hide  Versteckt dich auf der Livekarte (Man ist auch im Gm3 sichtbar, Befehl hält für immer)
  • /dynmap show  Zeigt dich wieder auf der Livekarte

 

Render

 

  • /dynmap render  Rendert das Quadrat in dem du stehst
  • /dynmap radiusrender <radius>  Rendert das Gebiet im angegeben Radius
  • /dynmap cancelrender <welt>  Bricht den Render in der gewählten Welt ab (Keine Welt angeben um die aktuelle abzubrechen

 

   ! Der Render läuft nur, wenn weniger als 8 Spieler auf dem Server sind. Wird der Wert überschritten, pausiert dieser solange.

 

Livemap Tags

 

  • /dmarker add id:<id> <label> icon:<icon> set:<set-id> Fügt einen Livemap Tag an deiner Postion hinzu
    • ID: Ist der einzigartige Name um diesen Tag identifizierbar machen zu können
    • Label: Ist der Name welcher auf der Karte gezeigt wird. Dieser kann beliebig oft vorkommen
    • Icon: Definiert welches Icon genommen werden soll. Eine Auswahl gibt's hier.
    • Set: Jedes Icon muss in ein Set gespeichert werden. Das ist wie ein Ordne

 

  • /dmarker listsets  Listet alle existierenden Sets auf
  • /dmarker addset id:<id> <label>  Erstellt ein neues Set mit ID und Namen
  • /dmarker movehere id:<id> set:<set>  Verschiebt den Marker zu deiner Position
  • /dmarker delete id:<id> set:<set>  Löscht einen Marker wieder

 

Eine Welt von der Karte verschwinden lassen

 

Befehle in dieser Reihenfolge eingeben um eine Weltkarte zu entfernen oder wieder hinzuzufügen:

  1. /dynmap pause
  2. /dmap worldset <welt> enabled:<true/false>
  3. /dynmap pause none
